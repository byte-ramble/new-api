## 为什么需要这个

OmniRouter 提供：
- **按量付费分组**：明码标价，您用多少付多少 —— **无公平使用限制**
- **包月 / 订阅分组**：固定费用，理论上不限调用 —— **必须有公平使用限制**

如果不限制订阅类用户，少数极端用户的高频调用会被你订阅费用 cover，导致：
- 我们亏损 → 服务质量下降 → 所有用户受影响
- 上游服务商认为是滥用 → 限速 / 封禁我们的整条线路

所以我们对**订阅类**和**包月套餐**实施公平使用规则。

## 2. 公平使用阈值

### 2.1 Codex 包月（¥60/月示例）

**正常使用范围**（绝大多数用户）：
- 24 小时内 ≤ **800 次** Codex 任务调用
- 单次调用 ≤ **1M token** prompt + completion
- 月内 ≤ **20K** 次累计调用

**触发降级**的情况：
- 24 小时内 > 800 次任务 → 自动降速（限速到 10 RPM）持续 1 小时后恢复
- 单次调用 > 1M token → 拒绝该次请求，提示拆分
- 月内 > 20K 次 → 月内剩余时间转按量计费

**触发封禁**的情况：
- 自动化脚本疑似刷流量（短时间 > 100 次 + 高重复 prompt）
- 同一 IP 多账户共享一个包月
- 倒卖包月给第三方

### 2.2 注册赠送（$1 例）
- 单账户限 $1（不重复发放）
- 不可与其它优惠叠加
- 30 天内未使用作废
- 仅限同一自然人 / 法人首次注册

### 2.3 多账户检测
我们通过以下信号识别多账户：
- 同一 IP 短时间多次注册
- 同一设备指纹（浏览器 + 屏幕 + 字体）
- 同一支付方式（信用卡 / 支付宝实名）
- 同一邮箱前缀（`+1`, `+test` 等技巧）
- 邀请链路高度同质（A 邀请 B/C/D 全用同一 IP 注册）

确认是同一主体的多账户，我们会：
- 合并奖励到主账户
- 没收虚增的额度
- 严重情形封禁所有关联账户

## 3. 客户端限制（重要）

### 3.1 Claude Code "CC" 分组
- **仅限 Anthropic 官方 Claude Code 客户端**
- 检测到 Cline、OpenCode、Aider 等第三方 IDE → **限速到 1 RPM**
- 反复违规 → 临时禁用该分组（可继续用其它分组）

**为什么**：CC 分组的低成本依赖于 Claude Code 客户端的特定调用模式（系统 prompt + 缓存优化）。第三方客户端会破坏假设、抬高成本、可能违反 Anthropic 的合作条款。

如果你坚持用第三方客户端用 Claude，请选 `claude-aws` 或 `claude-aws-q` 分组。

### 3.2 自研客户端
自研客户端 **欢迎**，只要：
- 通过标准 OpenAI / Anthropic / Gemini API 协议
- 不伪造 User-Agent 假装是官方客户端
- 不在受限分组（如 CC）使用

如有疑问，工单咨询。

## 4. 自动化与爬虫

### 4.1 允许
- 您自己应用的后端调用 OmniRouter（如 Web 应用、CI/CD pipeline、定时任务）
- 个人脚本调用合理频率（< 30 RPM 总）

- 用 OmniRouter 大规模爬取数据（高并发 + 24/7）
- 用 OmniRouter 做模型蒸馏 / 训练数据收集
- 高频压测（请联系我们提供专门的测试通道）

## 6. 违规处理流程

### 6.1 警告（轻度违规）
- 控制台站内消息提醒
- 邮件通知

### 6.2 临时降级（中度违规）
- 限速持续 1-24 小时
- 期间可继续使用，速度受限

### 6.3 临时封禁（重度违规）
- 账户冻结 24 小时 -7 天
- 期间无法登录、无法调用 API
- 余额保留

### 6.4 永久封禁 + 没收余额（严重违规）
- 多账户薅羊毛
- 倒卖账户
- 大规模违反 [用户协议](./tos.md)
- 涉嫌欺诈 / chargeback 滥用

### 6.5 申诉
所有违规处理均可申诉。
- 工单分类 "申诉"
- 提供您的解释 + 任何证据
- 5 个工作日内复核

## 7. 阈值会变

公平使用阈值不是法律承诺，是 **运营策略**。我们会根据：
- 上游成本变化
- 滥用情况
- 大多数用户的实际使用模式
随时调整阈值。

调整方式：
- 提前 7 天在 [{{DOMAIN}}/announcements]({{DOMAIN}}/announcements) 公告
- 注册邮箱通知
- 调整后第一次触发时显示提示

如调整使您不能继续使用，可申请按本月剩余天数比例退订。

## 8. 透明度

为了让您理解自己的使用情况：

- **触发降级** 时控制台显示当前限速 + 何时恢复
